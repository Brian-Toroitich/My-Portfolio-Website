
# Brian Toroitich | Personal Portfolio Website

## 🚀 Overview

This is a clean, responsive personal portfolio website built using only **HTML5** and **CSS3**. It serves as an online presence to showcase my background, skills, and projects as a passionate **Software Engineer** and **Tech Enthusiast**.

Designed with minimalism and usability in mind, this site makes it easy for potential collaborators, employers, or clients to understand my work, explore my professional journey, and get in touch.

---

## 🎯 Features

- **Single Page Layout** with smooth navigation.
- **Dedicated Sections**:
  - 🏠 Home
  - 👤 About
  - 🎓 Education
  - 💡 Interests
  - 💻 Projects
  - 📩 Contact Form
- **Custom Avatar Integration** in the header.
- **Font Awesome Icons** for visual engagement.
- **Downloadable CV** in the education section.
- **Contact Form** using `mailto:` for direct communication.
- **Fully Responsive Design** optimized for desktop and mobile.
